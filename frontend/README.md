# Frontend

React app built with Vite.

## Scripts

- `npm run dev` starts the development server.
- `npm run build` creates a production build.
- `npm run preview` serves the production build locally.
- `npm run lint` runs ESLint.

## Setup

1. Install dependencies with `npm install`.
2. Start the backend API before opening the frontend.
3. Run `npm run dev` to work on the client app.

## Notes

- Environment-specific values should be configured outside the repository.
- The frontend expects the backend and Socket.IO server to be available during development.
